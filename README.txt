To install navigate to 
C:\Users[username]\AppData\Roaming\godot\app_userdata\Atomcraft
and unzip this mod into that folder

OR

If on windows, run install-mod.bat

Make sure the new recipes are in Materials and Reactions
-- IF ASKED OVERWRITE FILES! THIS IS IMPORTANT IF UPDATING! --

ENJOY!

To uninstall, delete new files from C:\Users[username]\AppData\Roaming\godot\app_userdata\Atomcraft
in both Reactions and Materials.
Or (if on Windows) run uninstall-mod.bat
