# More-Burnables
Mod for Atomcraft which adds some "Lucky Blocks" which drop random materials of a certain type. 

# Instructions
    Windows: install-mod.bat installs mod from local files
    
    Linux: download source.tar.gz for the latest release and manually install that in your atomcraft godot folder

uninstall-mod.bat uninstalls mod based on local files

zip-script.bat creates zip for export
